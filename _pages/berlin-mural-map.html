Berlin Mural Map

A map showing the street murals in Berlin. Each mural is pinned on the map as image thumbnail, which can be clicked on for more info. The murals are bundled together in a stack when zoomed out.

Each mural can be geotagged and the metadata updated.

You can try the live demo.

You can find the specifications and code in this github repository.
