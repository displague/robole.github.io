Frequency Punchcard

A Frequency Punchcard is a grid with sized circles showing the frequency of an event.

It provides insight for planning, allowing someone to explore an overview of the schedule of an event when making a decision. It is used often for transport schedules to explore timetables.

You can find the specifications and code in this github repository.
