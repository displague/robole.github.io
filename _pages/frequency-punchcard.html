Frequency Punchcard

A Frequency Punchcard is a grid with sized circles showing the frequency of a trip/event.

It provides insight for planning to decide if there is a best/worst day to do a certain trip/event, and explore an overview of trips/events to make a schedule.

You can find the specifications and code in this github repository.
