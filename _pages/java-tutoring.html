Java Tutoring

I have taught a wide variety of students with different backgrounds, nationalties, and experience.

I emphasize analysing the requirements to write a design before writing any code, and adopt a test-driven development approach when students are at an appropriate level. I have compiled a collection of good example solutions to assignments to provide guidance for students and teachers.

Sample Projects

Grocery Warehouse Management System

Objective

Write a Graphical User Interface (GUI) or command-line application to manage customers, sales and inventory for a Grocery Warehouse.

Topics to know in order to complete the project

  1. Java syntax and programming basics
  2. How to implement Classes with instance variables and methods
  3. How to implement a design using OO Concepts such as: Polymorphism, Encapsulation, and Composition
  4. Abstract classes
  5. Interfaces
  6. Error Handling
  7. GUI Design: Swing Components and Layout Managers
  8. Event-driven programming (ActionListeners)
  9. Collections
  10. File I/O

Difficulty

Suitable for advanced students. This could be 20-40 hours of work when material is covered but it is hard to say exactly as the specification is quite verbose and has requirements that are counter-intuitive.
