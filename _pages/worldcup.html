⚽️ World Cup 🏆

Objective

  1. To create an appealing, responsive website that allows people to easily explore the teams and matches in the Fifa World Cup 2018.
  2. To make the data available through a REST API for developers! One love, yo!👩‍💻💓👨‍💻

Why?

The official Fifa website has a poor mobile user experience.😑📲 It can be improved upon generally. I didn't see any other single website that offered a good guide to the world cup, along with live scores, and results. I wanted to see if I could make a happy home for all of that.

Not many APIs for world cup data. This was addressed partially through this REST API by the people at Software for Good. It scrapes data from the Fifa website and has been made availabe for free. 🙌 It is well designed, but the performance is a bit sluggish. I ain't complaining, I'm just stating the fact, it is free as I mentioned! The data available is predominantly for matches, I will re-use some of it, and blend it with my own API.🍹

Finished Application🚀

You can try it!

It is hosted on heroku using a free account. The website sleeps 😴 after 30 minutes of inactivity. So, it may take 2 minutes for the website to appear for you. It's worth the wait! The performance is a bit wanting 🐢. Of course, when you pay, everything performs well. I'm open for donations for the next competition (Euro 2020)!
